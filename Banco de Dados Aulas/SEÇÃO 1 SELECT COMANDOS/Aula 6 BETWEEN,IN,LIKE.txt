BETWEEN-Pega um valor para encontrar o valor minimo e maximo

SELECT *
FROM Production.Product
WHERE ListPrice between 1000 and 1500;

SELECT *
FROM Production.Product
WHERE ListPrice NOT between 1000 and 1500;

SELECT *
FROM HumanResources.Employee
WHERE HireDate BETWEEN '2009/01/01'and '2010/01/01';

IN-Usado junto com WHERE serve para verificar se qualquer valor passado nas lista de valores comrrespondem a tal valor.
SELECT *
FROM Person.Person
WHERE BusinessEntityID IN (2,7,13);

SELECT *
FROM Person.Person
WHERE BusinessEntityID NOT IN (2,7,13);



LIKE-Vamos dizer q vc lembra uma parte de um nome de uma tal pessoa exemplo beni alguma coisa o LIKE serve pra te dar esse filtro caso vc queira encontrar determinado nome.
SELECT *
FROM Person.Person
WHERE FirstName like 'ovi%'; inicio do nome

SELECT *
FROM Person.Person
WHERE FirstName like '%to';  final do nome

SELECT *
FROM Person.Person
WHERE FirstName like '%essa%'; meio do nome
